# SAM OS

SAM OS is an experimental operating system being built from scratch as a structured systems engineering project.

## Overview

SAM OS focuses on developing a minimal, bootable operating system engine starting from low-level programming principles.

This project is being developed by:

- Aderiye Oluwasola Michael
- Donatus Michael

## Roadmap

The development follows a structured 100-day challenge:

- 60 days — C programming, memory management, systems fundamentals
- 40 days — Kernel architecture and bootloader implementation

## Version 0.1 Goals

- Minimal bootable engine
- No GUI
- Experimental internal platform

## Tech Stack

- C
- Assembly (planned)
- GCC
- QEMU (for testing)
- Linux development environment

## Status

Currently in foundational development phase.
